# 🧩 Sliding Puzzle Game

An **interactive sliding puzzle game** built using **React.js** that challenges users to solve a classic tile-based puzzle while tracking their moves in real time.  
The game includes **automatic puzzle-completion validation** to instantly detect when the puzzle is successfully solved.

---

## 🎮 Features

- 🧩 **Classic Sliding Puzzle Gameplay**  
  Drag and slide tiles to arrange them in the correct order.

- 🔢 **Move Tracking**  
  Tracks and displays the total number of moves made by the player.

- ✅ **Automatic Puzzle Completion Detection**  
  Instantly validates when the puzzle is solved correctly.

- ⚡ **Smooth & Responsive UI**  
  Optimized for seamless gameplay across different screen sizes.

---

## 🛠 Tech Stack

- ⚛️ **Framework:** React.js (Vite)  
- 💻 **Language:** JavaScript  
- 🎨 **Styling:** CSS

## 🏁 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/yogesh968/game
cd sliding-puzzle-game
```

### 2️⃣ Install dependencies
```bash
npm install
```

### 3️⃣ Run the development server
```bash
npm run dev
```

### 4️⃣ Build for production
```bash
npm run build
```
